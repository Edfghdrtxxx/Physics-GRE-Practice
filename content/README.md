# Content drop zone

Put the **Conquering the Physics GRE** markdown file here when it's ready, e.g.:

```
content/conquering-the-physics-gre.md
```

This folder is just the archive/staging spot. To get the content **into the site**,
open the app → **Library** → drag the file in (or "Choose a file…"). The text is
stored locally in your browser's IndexedDB; map its sections to topic portals and
they'll render on each portal's *Notes & reading* card with math typeset offline.

The structure-aware parser that turns the book's end-of-chapter problems into
practice questions is designed but waits for the real file — see `../docs/DESIGN.md` §3.
