# Content — generated question bank & book assets

Two **gitignored** folders live here (copyrighted book-derived content — never pushed
to the public repo; the rule is enforced in `.gitignore`):

- `bank/` — generated by the offline extraction pipeline from
  `docs/Conquering the Physics GRE (Yoni Kahn)/`:
  - `cpg-questions.js` → `PGRE.BOOK_QUESTIONS` (146 end-of-chapter problems, per topic,
    with worked solutions and per-choice distractor explanations)
  - `cpg-exams.js` → `PGRE.BOOK_EXAMS` (Sample Exams 1–3, 100 questions each — powers
    the mock-exam simulator's legacy mode)
  - `cpg-formulas.js` → `PGRE.BOOK_FORMULAS` (334 formula cards, strictly 1:1 with the
    book's numbered equations per its EQUATION INDEX)
- `book-assets/` — the figure JPEGs referenced by bank questions.

`js/bank.js` merges these with the 20 preview questions at load; the files are optional
(the site runs with empty placeholders if they're absent). If they're ever lost, re-run
the extraction pipeline against the book markdown — the site code contains no book
content itself.

The **Library** page (`#/library`) drag-drop import still works for any additional
markdown you want rendered in topic portals (stored in IndexedDB, searchable via
`#/search`).
